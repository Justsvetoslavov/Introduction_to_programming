# Задача 1.

**Конкатенация** на две цели неотрицателни числа `A` и `B` наричаме числото `A ∘ B`, което се получава, като към цифрите на `A` се долепят цифрите на `B`.

### Примери
```
123 ∘ 431 = 123431
123 ∘ 0 = 1230
```

Да се напише функция, която въвежда две **неотрицателни цели числа** `N` и `K` и извежда дали съществуват **положителни цели числа** `A` и `B` (A > 0 и B > 0), такива че: `N = A ∘ K ∘ B`

Приема се, че `N` и `K` са числа с **до приблизително 18 цифри**, които могат да бъдат представени в стандартен беззнаков целочислен тип.

### Примери

| Вход | Изход | Обяснение |
|------|--------|------------|
| N = 1234, K = 4 | False | `K` е в края ⇒ няма място за `B` |
| N = 1234, K = 2 | True | `N = 1 ∘ 2 ∘ 34` |
| N = 1234, K = 23 | True | `N = 1 ∘ 23 ∘ 4` |

В решението **не се допуска използването на масиви, низове или структури от данни от STL.**
