# Увод. Примитивни типове данни. Преобразуване между типове. Оператори.

## Термини

- Програма - редица от инструкции, водеща до решаване на определена задача.
- Компютърна програма - представлява код или алгоритъм, който може да бъде "разбран", разчетен и изпълнен от процесора или най-общо казано от компютъра.
- Алгоритъм - поредица от инструкции/стъпки за решаване на даден проблем.
- Програмиране -  компютърна наука и дял от математиката-информатиката, която се занимава с алгоритмите, методите и средствата за създаване на компютърни програми.
- Език за програмиране - изкуствен език, предназначен за задаване на команди, които искаме компютърът да прочете, обработи и изпълни.
- Компилатор -  превежда кода от програмен език на машинен код.
- Среди за разработка/Среди за програмиране (IDE = Integrated Development Environment) - съвкупност от традиционни инструменти за разработване на софтуерни приложения.

## Стандартен вход&изход

- Input/Output in C++ - под формата на поредица от байтове или по-популярните streams.
- Input stream - Ако посоката на потока от байтове е от устройството (например клавиатура) към главната памет, тогава този процес се нарича вход.
- Output stream - Ако посоката на потока от байтове е обратното, т.е. от главната памет към устройството (например екран), тогава този процес се нарича изход.

## Променливи 

Деклaрация на променлива:
```c++
<тип> <име> = <стойност>;
```

### Примитивни типове:**
- Целочислен тип (int, short, long, long long).
- Числа с плаваща запетая (double, float).
- Булев тип (bool).
- Символен тип (char).

### Unsigned типове:
- За целочислени типове можем да използваме unsigned вариант, който съхранява само положителни числа:
  - `unsigned int`, `unsigned short`, `unsigned long`, `unsigned long long`
  - `unsigned char` - често се използва за байтове

### Размери на типовете (зависи от системата, типични стойности):
| Тип | Размер (байтове) | Диапазон (signed) |
| :---: | :---: | :---: |
| bool | 1 | true/false |
| char | 1 | -128 до 127 |
| short | 2 | -32,768 до 32,767 |
| int | 4 | -2,147,483,648 до 2,147,483,647 |
| long | 4 или 8 | зависи от системата |
| long long | 8 | -9,223,372,036,854,775,808 до 9,223,372,036,854,775,807 |
| float | 4 | ±3.4e±38 (7 цифри точност) |
| double | 8 | ±1.7e±308 (15 цифри точност) |

> **Забележка:** Можете да проверите размера на тип с оператора `sizeof(тип)` или `sizeof(променлива)`.

Съществуват и други примитивни типове, които ще се разгледат по-напред в курса.

```c++
int N = 50;          // N е целочислена променлива със стойност 50
double PI = 3.14159; // PI е променлива от тип double със стойност 3.14159
bool bOk = 0;        // bOk е булева променлива със стойност 0, т.е. false
char symbol = 'N';   // symbol е символна променлива със стойност 'N'
```

### Име на променливи
- Могат:
  - да съдържат малки букви.
  - да съдържат главни букви.
  - да съдържат цифри.
  - да съдържат долни черти.
- **НЕ** могат:
  - да са запазени думи(int,main,return и т.н).
  - да започват с цифра.
  - да съдържат интервали.

## Преобразуване между типовете

### Имплицитно (автоматично) преобразуване

Преобразуването може да се извърши:

**Без загуба на информация:**
```c++
bool b = true;
int a = b; // 1

int n = 4; 
double m = n; // 4.0

char c = 'A';
int code = c; // 65 (ASCII код на 'A')
```

**Със загуба на информация:**
```c++
int a = 10;
bool b = a; // 1 (true) - всяко ненулево число става true

double PI = 3.14;
int truncated = PI; // 3 (отрязва се дробната част)

int large = 300;
char small = large; // Undefined behavior - 300 не се побира в char!
```

### Explicit (явно) преобразуване (casting)

```c++
// C-style cast
int a = (int)3.14;  // 3

// C++ style cast (препоръчителен)
int b = static_cast<int>(3.14);  // 3

// При деление на цели числа
int x = 5;
int y = 2;
double result1 = x / y;                    // 2.0 (int деление, след това конвертиране)
double result2 = (double)x / y;            // 2.5 (конвертира x в double)
double result3 = static_cast<double>(x) / y; // 2.5 (препоръчителен начин)
```
 
## Oператори

#### Аритметични оператори:

| Оператор | Значение | Пример | Резултат |
| :---: | :---: | :---: | :---: |
| + | Събиране | 5 + 2 | 7 |
| - | Изваждане | 5 - 2 | 3 |
| * | Умножение | 5 * 2 | 10 |
| / | Деление | 5 / 2 | 2 |
| % | Деление с остатък | 5 % 2 | 1 |

**Важно за деление:**
- Деление между цели числа връща цяло число (отрязва се дробната част)
- Ако поне едно от числата е `double` или `float`, резултатът е с плаваща запетая
```c++
int a = 5 / 2;      // 2 (integer деление)
double b = 5.0 / 2; // 2.5
double c = 5 / 2.0; // 2.5
```

#### Инкремент и декремент оператори:

| Оператор | Значение | Пример | Обяснение |
| :---: | :---: | :---: | :---: |
| ++ | Префиксен инкремент | ++a | Увеличава a с 1, после връща новата стойност |
| ++ | Постфиксен инкремент | a++ | Връща старата стойност, после увеличава a с 1 |
| -- | Префиксен декремент | --a | Намалява a с 1, после връща новата стойност |
| -- | Постфиксен декремент | a-- | Връща старата стойност, после намалява a с 1 |

```c++
int x = 5;
int y = ++x; // x = 6, y = 6 (първо увеличава, после присвоява)

int a = 5;
int b = a++; // a = 6, b = 5 (първо присвоява, после увеличава)
```

#### Оператори за сравнение (винаги връщат истина/лъжа - bool):

| Оператор | Значение | Пример | Резултат |
| :---: | :---: | :---: | :---: |
| < | По-малко | 1 < 2 | true |
| > | По-голямо | 1 > 2 | false |
| <= | По-малко или равно | 1 <= 2 | true |
| >= | По-голямо или равно | 2 >= 2 | true |
| == | Равно на | 1 == 2 | false |
| != | Не е равно на | 1 != 2 | true |

> **Важно:** За сравнение на равенство използваме `==`, а не `=` (което е присвояване)!

#### Логически оператори:

| Оператор | Значение | Пример | Резултат |
| :---: | :---: | :---: | :---: |
| && | Логическо **И** (AND) | (5<2) && (5>3) | False |
| \|\| | Логическо **ИЛИ** (OR) | (5<2) \|\| (5>3) | True |
| ! | Логическо **НЕ** (NOT) | !(5<2) | True |

**Short-circuit evaluation (късо съединение):**
- При `&&`: Ако лявата част е `false`, дясната **не се оценява**
- При `||`: Ако лявата част е `true`, дясната **не се оценява**

```c++
int x = 0;
bool result = (x != 0) && (10 / x > 2); // Безопасно! Не дели на 0
                                          // защото (x != 0) е false
```

#### Оператори за присвояване:
| Оператор | Пример | Eквивалентен |
| :---: | :---: | :---: |
| += | m += 10 | m = m + 10 |
| -= | m -= 10 | m = m - 10 | 
| *= | m *= 10 | m = m * 10 | 
| /= | m /= 10 | m = m / 10 |
| %= | m %= 10 | m = m % 10 |

Всеки оператор има:
- Приоритет
- Асоциативност
- Позиция на оператора спрямо аргумента - префиксен, инфиксен и суфиксен. <br />
